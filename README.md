# I am the Earth

太陽系を舞台にした対戦型アクションゲーム。プレイヤーは地球を操作し、月を発射して他のプレイヤーを攻撃します。

## ゲームの特徴

- 2〜4人対戦
- 太陽の重力を利用した戦略的なゲームプレイ
- 水星から水を補給、金星を避けるなどの要素

## セットアップ方法

### 必要なもの
- Processing 4.x (https://processing.org/download)
- Sound ライブラリ（サウンドを鳴らす場合）

### Soundライブラリのインストール

**詳細な手順は `SOUND_SETUP.md` を参照してください。**

簡単な手順:
1. Processing IDE を開く
2. `Sketch` → `Import Library...` → `Add Library...`
3. "Sound" を検索してインストール
4. Processing を再起動

### 実行方法

1. Processing IDE でこのフォルダを開く
2. `src/main.pde` を開く
3. 実行ボタン（▶）を押す

## 操作方法

### タイトル画面
- **Q** / **P** / **Z** / **M**: プレイヤー1〜4が参加
- **スペースキー**: ゲーム開始（2人以上必要）
  - 注: 吹き出しで親切に説明されています（「宇宙のスペース」ではなく「キーボードのスペースキー」です！）

### ゲーム中
- 各プレイヤーは自分のキーを**長押し**してチャージ、**離す**と月を発射
  - プレイヤー1: **Q**
  - プレイヤー2: **P**
  - プレイヤー3: **Z**
  - プレイヤー4: **M**
- チャージ中に1周するごとに月が成長（最大3段階）
- **B**: サウンドON/OFF（注: 音声ミュート状態は保存されません）

### ゲームオーバー
- **R**: タイトル画面に戻る

## ゲームルール

- 他のプレイヤーの地球に月をぶつけると脱落させられる
- 太陽に近づきすぎると燃える
- 水星に月をぶつけると水を補給できる（水星に水がある時のみ）
- 金星に月をぶつけると月がリセットされる
- 最後まで生き残ったプレイヤーが勝利

## 特殊演出

- **リザルト画面**: 勝利したプレイヤーの地球をカメラが追従＆ズームアップ（2.5倍）
- **BGM**: タイトル、バトル、リザルトで自動的に切り替わる（音量は控えめに設定）
- **画面振動**: 衝突時にヒットストップと画面揺れ
- **パーティクル**: 各種エフェクト（爆発、水しぶきなど）

## 配布方法

### 方法1: Soundライブラリを含めて配布（推奨）
詳細は `SOUND_SETUP.md` を参照

### 方法2: Soundライブラリなしで配布
プロジェクトフォルダを配布し、各自でSoundライブラリをインストールしてもらう

## トラブルシューティング

- **音が鳴らない**: `SOUND_SETUP.md` を確認してSoundライブラリをインストール
- **"SoundFile does not exist" エラー**: Soundライブラリが未インストールまたは未読み込み
