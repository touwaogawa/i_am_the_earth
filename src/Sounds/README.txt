【音声ファイルの配置について】

このフォルダに以下の音声ファイルを配置してください：

1. launch.wav または launch.mp3 - 月発射音
2. hit.wav または hit.mp3 - 衝突音
3. explosion.wav または explosion.mp3 - 爆発音
4. burn.wav または burn.mp3 - 燃焼/蒸発音
5. splash.wav または splash.mp3 - 水しぶき音
6. gravity.wav または gravity.mp3 - 引力効果音

※現在は各ファイル名.txtファイルがプレースホルダーとして配置されています
※実際の音声ファイルを配置したら、.txtファイルは削除してください
※SoundManager.pdeを修正して、実際の音声ファイルを読み込むようにしてください
